1. Information We Collect

The Combinations bot collects only the minimum amount of data required to operate properly.

1.1 Automatically Collected Data

  • Discord User IDs
  • Discord Server IDs
  • Discord Channel IDs (for feature-specific usage)
  • Command usage logs (timestamp, command name, etc.)

The bot does NOT store:

  • Message contents
  • IP addresses
  • Email addresses
  • Real names

The bot never reads, stores, or analyzes private messages (DMs).

2. Premium-Related Data

If a user purchases Premium (priced at 5 USD), the following data may be collected:

  • Transaction ID
  • Payment status
  • Discord User ID linked to the Premium access

All financial data such as credit card information is handled externally by third-party payment processors (e.g., PayPal, Stripe). No financial data is stored by the bot.

3. How Data Is Used

The collected data is used only for the following purposes:

  • Providing bot features
  • Verifying Premium access
  • Improving performance
  • Debugging and preventing abuse

We do not use your data for advertising, profiling, selling, or sharing with unauthorized third parties.

4. Data Retention

We store data only as long as necessary for the bot's functionality or as required by law. Users may request data deletion at any time by contacting the developer.

5. Data Sharing

Your information will never be sold, shared, or disclosed to any third parties, except:

  • Payment processors (only for handling Premium purchases)

6. Security

All stored data is protected using reasonable security measures. However, no system is completely secure, and the developer cannot guarantee absolute protection.

7. User Rights

You have the right to:

  • Request a copy of the data stored about you
  • Request deletion of your stored data
  • Withdraw consent by discontinuing use of the bot

8. Contact

For questions, concerns, or data deletion requests, contact the developer through Discord or email.
