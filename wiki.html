PermiX Dokumentáció

A PermiX egy ultrakönnyű, SQL-alapú jogosultságkezelő rendszer, amit kifejezetten nagy forgalmú szerverekre terveztünk a lag minimalizálása érdekében.

Letöltés (SpigotMC) Verzió: v1.4.2
Info: A plugin működéséhez szükséges a PlaceholderAPI és egy MySQL adatbázis kapcsolat.

Telepítés

  1. Töltsd le a PermiX.jar fájlt.
  2. Húzd be a szervered /plugins mappájába.
  3. Indítsd újra a szervert, majd nyisd meg a config.yml fájlt az adatbázis adatok megadásához.

Parancsok és Jogok

Parancs Jog (Permission) Leírás
/permix user set permix.admin Rang beállítása egy játékosnak.
/permix group add permix.admin Új rang létrehozása.
/permix reload permix.reload Config fájlok újratöltése.
Figyelem: Soha ne add meg a permix.* jogot olyan játékosnak, akiben nem bízol 100%-ig!

Konfiguráció

Íme egy példa a groups.yml beállítására:

config.yml 
groups:
  default:
    prefix: "&7[Játékos] "
    default: true
    permissions:
      # Alapvető jogok
      - essentials.spawn
      - essentials.msg
  admin:
    prefix: "&c[Admin] "
    permissions:
      - "*"
Tipp: Használd a /permix gui parancsot a rangok vizuális szerkesztéséhez játékon belül!

Placeholderek

Ezeket használhatod a Tab listán vagy a Scoreboardon:

  • %permix_rank% - Játékos rangjának neve
  • %permix_prefix% - Játékos prefixe
  • %permix_suffix% - Játékos suffixe
